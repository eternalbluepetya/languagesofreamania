Imperfective aspect

The ‘imperfective’ aspect refers to ongoing actions, such as I am learning and habitual actions, such as I learn (something new every day).

Mandasian uses an affix for imperfective:

Imperfective Suffix -ɔ
kagpiaw /kagˈpiɔ/
learns/is learning



Perfect aspect

The perfect aspect in English is exemplified in ‘I have read this book’, which expresses an event that took place before the time spoken but which has an effect on or is in some way still relevant to the present.

Mandasian uses an affix for the perfect aspect:

Perfect If ends with vowel: Suffix -ŋ
Else: Suffix -aŋ
kagping /kagˈpiŋ/
have learned

Else: Suffix -aŋ

Else: Suffix -an
Adjective → verb (to make something [adj]) = Suffix -o
Noun → adjective (having the quality of [noun]) = Suffix -is

Else: Suffix -ak
