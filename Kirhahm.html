Perfect aspect

The perfect aspect in English is exemplified in ‘I have read this book’, which expresses an event that took place before the time spoken but which has an effect on or is in some way still relevant to the present.

Kirhahm uses an affix for the perfect aspect:

Perfect If starts with vowel: Prefix ts-
Else: Prefix tsé-
tséviltn /ˈtséviltn/
have learned

Derivational morphology

Adjective → adverb = Suffix -é
Adjective → noun (the quality of being [adj]) = If ends with vowel: Suffix -n
Else: Suffix -en
Adjective → verb (to make something [adj]) = If ends with vowel: Suffix -ɾì
Else: Suffix -aɾì
Noun → adjective (having the quality of [noun]) = If starts with vowel: Prefix ʤ-
Else: Prefix ʤa-
Noun → adjective relating to noun (e.g. economy → economic) = If ends with vowel: Suffix -ndó
Else: Suffix -òndó
Noun to verb = Suffix -o
